## Password-Strength-Checker

“A Python tool that scores password strength based on length, digits, and uppercase letters.”

This project analyzes a password and assigns a strength score from 0 to 5.
It checks for:

Minimum length
Strong length (12+ characters)
Contains digits
Contains uppercase letters
